# Consolidamento sessione 2

## Argomenti

### Funzioni

Prepara una lezione sulle funzioni.
Dovrai preparare le slide e gli esempi pratici necessari per fare una lezione che riassuma i
punti chiave di questo argomento ai tuoi compagni. La presentazione durerà 10 minuti.
Alla fine, le tue slide saranno parte del "Compendium di classe ###", che potrà essere condiviso
coi compagni per prepararsi alle verifiche.

Usa queste domande come linee guida per le tue slide:

- Cosa è una funzione?
- Perchè è utile?
- Quando è opportuno usare una funzione?
- Qual è la differenza tra una funzione tradizionale e un'arrow function?
- Qual è la sintassi di una funzione?
- Qual è la differenza tra parametro e argomento?
- E' obbligatorio avere un return?
- Una funzione "vede" le variabili al suo esterno?
- cosa si intende per funzione anonima?
- cosa si intende per funzione "di callback"?
- cosa si intende per hoisting
- cosa è un'arrow function e quali sono le sue caratteristiche principali?
- qual è la sintassi di un'arrow function?
- cosa si intende per "return implicito" in un'arrow function?

### Manipolazione del DOM

Prepara una lezione sulla manipolazione del DOM in JavaScript.
Dovrai preparare le slide e gli esempi pratici necessari per fare una lezione che riassum le cose più importanti e utili.
Alla fine, le tue slide saranno parte del "Compendium di classe ###", che potrà essere consultato in futuro

Usa queste domande come linee guida per le tue slide:

- Cosa è il DOM?
- Cosa significa "manipolazione del DOM"?
- Come si seleziona un elemento dal DOM?
- Cosa è getElementById()?
- Cosa è querySelector()?
- Qual è la differenza tra getElementById() e querySelector()?
- Come si selezionano più elementi?
- Cosa è querySelectorAll()?
- Come si modifica il contenuto di un elemento?
- Cosa è il metodo innerHTML?
- Cosa è il metodo innerText?
- Qual è la differenza tra innerHTML e innerText?
- Come si modificano le classi di un elemento?
- Come si accede a classList?
- Come si modificano gli stili direttamente?
- Come si ascolta gli eventi del DOM?
- Cosa è addEventListener()?
