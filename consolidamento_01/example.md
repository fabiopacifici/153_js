## Le variabili in JavaScript

Una variabile server per conservare nella memoria del computer dei dati:

- numeri
- strighe
- boolean

Le variabili di dichiarano usado delle parole chiavi:

- let, const (modene)
- var (obsoleta)

Le variabili deveono avere nomi descrittivi detti identificatiori:

example

```js
const age = 50
let name = 'Mario'

```
